Perguntas Frequentes

Como cadastrar um novo produto?

Clique para ver instruções detalhadas sobre como adicionar produtos ao sistema.

  1. Acesse o menu "Produtos" no sistema
  2. Clique no botão "Novo Produto"
  3. Preencha os campos obrigatórios (nome, preço, estoque)
  4. Adicione informações opcionais como descrição e categoria
  5. Clique em "Salvar Produto"

Dica: Configure o estoque mínimo para receber alertas quando o produto estiver acabando.

Como criar um pedido?

Clique para ver instruções detalhadas sobre como criar pedidos no sistema.

  1. Acesse o menu "Pedidos" no sistema
  2. Clique no botão "Novo Pedido"
  3. Selecione ou cadastre o cliente
  4. Adicione os produtos desejados
  5. Configure a mesa (se aplicável)
  6. Adicione observações se necessário
  7. Clique em "Criar Pedido"

Dica: Você pode alterar o status do pedido conforme ele é processado.

Como gerenciar o estoque?

Clique para ver instruções sobre controle de estoque.

  1. Acesse "Produtos" para ver o estoque atual
  2. Use o botão de ajuste de estoque para adicionar/remover itens
  3. Configure estoque mínimo para receber alertas
  4. Monitore produtos com baixo estoque no dashboard

Dica: O sistema mostra automaticamente produtos com estoque baixo no dashboard.

Como usar o controle de mesas?

Clique para ver instruções sobre o sistema de mesas.

  1. Acesse o menu "Mesas" no sistema
  2. Visualize o status de todas as mesas
  3. Clique em uma mesa para alterar seu status
  4. Use as ações para visualizar, editar ou excluir mesas
  5. Configure a capacidade e localização das mesas

Dica: As mesas podem estar Livres, Ocupadas ou Reservadas.

Como gerar relatórios?

Clique para ver instruções sobre relatórios e análises.

  1. Acesse o menu "Relatórios" no sistema
  2. Selecione o período desejado
  3. Escolha o tipo de relatório
  4. Clique em "Gerar Relatório"
  5. Use "Exportar" para baixar os dados

Dica: Os relatórios mostram estatísticas importantes do seu negócio.

Como usar o controle de caixa?

Clique para ver instruções sobre o sistema financeiro.

  1. Acesse o menu "Caixa" no sistema
  2. Visualize o saldo atual e estatísticas
  3. Use "Registrar Pagamento" para entradas
  4. Use "Retirada" ou "Despesa" para saídas
  5. Monitore todas as movimentações

Dica: Mantenha o controle financeiro sempre atualizado.
