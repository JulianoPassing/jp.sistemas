========================================
    JP-Cobranças NOVO - Instruções Rápidas
========================================

🎯 COMO USAR O SISTEMA NOVO:

1. INICIAR O SISTEMA:
   - Windows: Execute "install.bat"
   - Linux/Mac: Execute "./install.sh"
   - Ou manualmente: 
     cd backend && npm install && npm start

2. CONFIGURAR BANCO DE DADOS:
   - Instale o MariaDB/MySQL
   - Execute: mysql -u root -p < ../database_config.sql
   - Configure as credenciais em backend/db.js se necessário

3. ACESSAR:
   - Backend: http://localhost:3001
   - Frontend: Abra frontend/index.html no navegador

4. NAVEGAÇÃO:
   - Dashboard: Página principal com resumo
   - Clientes: Cadastro e gerenciamento de clientes
   - Empréstimos: Controle de empréstimos
   - Cobranças: Controle de pagamentos
   - Atrasados: Clientes em inadimplência

🔧 CONFIGURAÇÃO DO BANCO:

1. Instale o MariaDB/MySQL
2. Execute: mysql -u root -p < database_config.sql
3. O banco será criado automaticamente com dados de exemplo
